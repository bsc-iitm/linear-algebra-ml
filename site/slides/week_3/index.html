    ::: {.columns align=left}

    ::: {.column width="50%"}




    $$ N(X) = N(X^TX) $$ :::

    ::: {.column width="50%"}

    Proof

    If \(\theta \in N(X)\):

    • \(X\theta =0\)
    • \(X^TX \theta = 0\)

    If \(\theta \in N(X^T X)\)

    ::: incremental

    • \(X^TX \theta = 0\)
    • \(\theta^T X^TX \theta = 0\)
    • \((X \theta)^T (X \theta) = 0\)
    • \(X \theta = 0\)

    If \(\text{rank}(X) = n\), \(X^TX\) is invertible

    :::

    ::: {.column width="50%"}

    Proof

    If \(\text{rank}(X) = n\):

    ::: incremental

    • \(\text{nullity}(X) = 0\)
    • \(\text{nullity}(X^TX) = 0\)
    • \(\text{rank}(X^TX) = n\)
    • \(X^TX\) is full rank, hence invertible
